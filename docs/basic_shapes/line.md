#### 线段 line

用直尺把两点连接起来，就得到一条线段。线段长就是这两点间的距离。 在连接两点的所有线中，线段最短。简称为**两点之间线段最短**。 

只要确定线段的两个定点的位置，就可以确定一条线段的位置。 

```javascript
line(x1, y1, x2, y2, *lineWidth, *color)
/**************************
  x1 - 起点x坐标
  y1 - 起点y坐标
  x2 - 终点x坐标
  y2 - 终点y坐标
  lineWidth - 线宽，默认为1
  color - 颜色，默认为orange
**************************/
```

如果我们想要实现更多复杂的效果，那么我们就需要使用Line对象，详见Line章节。
