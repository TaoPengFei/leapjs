## 补充图形

#### 矩形

##### 1. 创建一个叫做 r 的矩形

```javascript
var r = new Rectangle(150,200,50,100);
```

##### 2. 改变r的属性

* 改变r的左上角x坐标：`r.x`
* 改变r的左上角y坐标：`r.y`
* 改变r的宽度：`r.width`
* 改变r的高度：`r.height`
* 改变r的填充颜色：`r.fillStyle`
* 改变r的边框颜色：`r.strokeStyle`
* 改变r的边框宽度：`r.lineWidth`

##### 3. 用不同方法画出r

* 画出带边框的矩形：`r.draw()`
* 画出不带边框的矩形：`r.fill()`
* 画出矩形的边框：`r.stroke()`

#### 线段

##### 1. 创建一个叫做 l 的矩形

```javascript
var l = new Line(150,200,50,100);
```

##### 2. 改变l的属性

- 改变l起点的x坐标：`l.x1`
- 改变l起点的y坐标：`l.y1`
- 改变l终点的x坐标：`l.x2`
- 改变l终点的y坐标：`l.y2`
- 改变l的线条颜色：`l.strokeStyle`
- 改变l的线条宽度：`l.lineWidth`

##### 3. 画出l

- 画出线段：`r.draw()`

#### 三角形

##### 1. 创建一个叫做 t的矩形

```javascript
var t = new Triangle(150,200,50,100,200,200);
```

##### 2. 改变t的属性

- 改变t第一点的x坐标：`t.x1`
- 改变t第一点的y坐标：`t.y1`
- 改变t第二点的x坐标：`t.x2`
- 改变t第二点的y坐标：`t.y2`
- 改变t第三点的x坐标：`t.x3`
- 改变t第三点的y坐标：`t.y3`
- 改变t的填充颜色：`t.fillStyle`
- 改变t的边框颜色：`t.strokeStyle`
- 改变t的边框宽度：`t.lineWidth`

##### 3. 用不同方法画出t

- 画出带边框的三角形：`t.draw()`
- 画出不带边框的三角形：`t.fill()`
- 画出三角形的边框：`t.stroke()`
